# Opera Proxy Checker

Программа-чекер для проверки работы бинарного файла opera-proxy и публикации результатов на веб-странице.

## Функциональность

- Проверяет работоспособность бинарного файла opera-proxy
- Запускает opera-proxy с заданными параметрами
- Тестирует соединение через прокси
- Публикует результаты на веб-странице
- Сохраняет историю проверок

## Установка на Keenetic router с Entware

### Подготовка

1. Убедитесь, что у вас установлен Entware на вашем маршрутизаторе Keenetic

### Установка

1. Скачайте предварительно собранный бинарный файл для ARM64:

```bash
wget https://github.com/yourusername/opera-proxy-checker/releases/download/v1.0/opera-proxy-checker.linux-arm64
chmod +x opera-proxy-checker.linux-arm64
```

2. Также скачайте бинарный файл opera-proxy для ARM64 (если еще не установлен):

```bash
wget https://github.com/Snawoot/opera-proxy/releases/latest/download/opera-proxy.linux-arm64
chmod +x opera-proxy.linux-arm64
```

3. Запустите чекер:

```bash
./opera-proxy-checker.linux-arm64 -binary ./opera-proxy.linux-arm64 -test-url http://httpbin.org/ip -interval 5m -web-port 8080
```

### Параметры запуска

- `-binary`: путь к бинарному файлу opera-proxy (по умолчанию "./opera-proxy")
- `-test-url`: URL для проверки прокси-соединения (по умолчанию "http://httpbin.org/ip")
- `-interval`: интервал между проверками (по умолчанию "5m")
- `-web-port`: порт для веб-интерфейса (по умолчанию "8080")

### Веб-интерфейс

После запуска программа запускает веб-сервер, который доступен по адресу:

```
http://router-ip:8080
```

На этой странице вы можете видеть результаты последних проверок.

## Архитектура

Программа состоит из следующих компонентов:

1. **Checker** - основная логика проверки работы opera-proxy
2. **WebServer** - веб-сервер для отображения результатов
3. **Result** - структура данных для хранения результатов проверки

## Результаты проверки

Каждая проверка сохраняет следующую информацию:

- Время проверки
- Статус (успех/ошибка)
- Время отклика
- Публичный IP (если доступен)
- Сообщение об ошибке (если произошла ошибка)
